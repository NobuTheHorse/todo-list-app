# ReactJS Todo List App

This is a simple React Todo List App that lets users add, complete, and delete todo items.

## Features
- Add new todo items
- Toggle items as completed
- Delete items
- State managed using React `useState`
- Hosted on Vercel (or GitHub Pages)
